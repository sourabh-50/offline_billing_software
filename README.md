# Omsai Billing Software 🌟

A professional, high-performance Windows desktop application for mobile retail shops to manage GST billing and inventory.

## ✨ Key Features
- **Modern UI/UX**: Built with CustomTkinter for a sleek, premium experience.
- **Dynamic Dashboard**: Real-time sales statistics with daily, weekly, and monthly filters.
- **GST Billing**: Automated PDF invoice generation with professional branding.
- **Multi-Firm Support**: Manage multiple shops within a single application.
- **Hardware-Locked Activation**: Anti-piracy system that binds the software to the client's machine.
- **Data Maintenance**: Built-in tools for data cleanup, Excel exports, and automated backups.

## 🛠️ Technology Stack
- **Python 3.x**
- **CustomTkinter** (GUI)
- **SQLite3** (Database)
- **ReportLab** (PDF Generation)
- **PyInstaller** (Standalone Executable)

## 📦 How to Build
To generate the standalone executable:
1. Install dependencies: `pip install -r requirements.txt`
2. Run build: `pyinstaller MobileShopBiller.spec`

## 🛡️ Distribution
Use the included `installer_script.iss` with **Inno Setup** to create a professional `Setup.exe` for your clients.
